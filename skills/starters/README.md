# Starter Templates

**Code you can modify.** These aren't demos to watch — they're starting points for your competition code.

| Template | Difficulty | What It Does |
|----------|-----------|-------------|
| `bulldoze_to_basket.py` | ⭐ Easy | Drive at a basket, push blocks along the way |
| `navigate_to_tag.py` | ⭐⭐ Medium | Go to any AprilTag, strafe around barriers using sonar |
| `color_sort_one.py` | ⭐⭐⭐ Hard | Pick up one block, deliver to matching color basket |
| `full_auto.py` | ⭐⭐⭐⭐ Expert | Complete loop: find, grab, deliver, repeat until time runs out |

## How to Use

1. Copy a template to your own file: `cp bulldoze_to_basket.py my_strategy.py`
2. Read the code — comments explain every section
3. Modify the `# CUSTOMIZE` sections
4. Test on the practice field
5. Iterate!

## Tips

- Start with `bulldoze_to_basket.py` — scores points with zero arm skill
- Graduate to `color_sort_one.py` when you've got pickup working
- `full_auto.py` is the endgame — modify it to win
- Ask GenAI to help: "Modify this template to prioritize red blocks"
